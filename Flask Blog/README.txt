## Exercise Description
  This exercise focuses on using Flask in creating a simple blog.
  The data provided for in the blog came from another programming
  exercise, PSE Market Cap Charts, wherein I used Matplotlib.
  The overall framework of the blog is a simple implementation in
  Flask. The HTML, CSS and JS add ons were sourced from Bootstrap.
  
## Insights
  This exercise helped me understand how to use Flask and the concept of
  how the framework operates, which is very simple and intuitive to the
  developer. It's a great starting point for web development in Python.
